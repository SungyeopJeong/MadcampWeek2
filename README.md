## **Devil**

- developers village 라는 뜻으로 개발자들 간 네트워킹 활성화
- 스터디(app, frontend, backend, etc) 를 모집하고 참여가능
- 스터디에 가입하면 해당 스터디 실시간 채팅방 이용가능

### 🧑‍💻 팀원

- 백승효(숙명여대 20)
- 정성엽(KAIST 22)

### ⚡ 개발 환경

- **FE** - flutter
- **BE** - nodejs(express), mysql
